## Descripción General:

Esta tarea implica el uso de Sage para realizar cálculos matemáticos específicos. A continuación, se proporciona una breve descripción de cada pregunta y las respuestas correspondientes utilizando Sage:

### 1. Calcula el floor y el ceiling de -3.56
   - Utiliza las funciones `floor()` y `ceil()` para calcular el valor entero más grande menor o igual a -3.56 (floor) y el valor entero más pequeño mayor o igual a -3.56 (ceiling).

### 2. Calcula el logaritmo en base e de 100, calcula el logaritmo en base 10 de 100, después calcula el cociente.
   - Utiliza la función `log()` para calcular el logaritmo en base e y en base 10 de 100. Luego, obtén el cociente de ambos logaritmos.

### 3. Calcula el logaritmo en base 2 de 64.
   - Utiliza la función `log()` para calcular el logaritmo en base 2 de 64.

### 4. Compara $e^iπ$ con una aproximación numérica, usando pi.n().
   - Evalúa la expresión matemática $e^{i\pi}$ y compara con su aproximación numérica utilizando `pi.n()`.

### 5. Calcula sin (π/2), cot (π/4) y csc (π/16).
   - Aplica las funciones trigonométricas `sin()`, `cot()`, y `csc()` para calcular los valores correspondientes para los ángulos dados.

## Ejecución del Código:
Para ejecutar el código proporcionado en las respuestas, asegúrate de tener Sage instalado en tu entorno de desarrollo. Luego, copia y pega el código en la interfaz de Sage y observa los resultados obtenidos.
